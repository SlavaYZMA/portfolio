# Квантовые портреты

Добро пожаловать в мир, где личность — это суперпозиция состояний, а наблюдение определяет образ. Проект использует алгоритмы для трансформации портретов в динамические системы, вдохновлённые уравнением Шрёдингера.

## Концепция
Каждый пиксель — это квантовый объект, чья волновая функция схлопывается под влиянием зрителя. Это метафора текучей идентичности в цифровую эпоху.

## Техническая реализация
- Алгоритмы: Симуляция волновой функции с QMsolve.
- Интерактивность: Курсор пользователя как инструмент коллапса.

# Quantum Portraits

Welcome to a world where identity is a superposition of states, shaped by observation. This project transforms portraits into dynamic systems inspired by the Schrödinger equation.

## Concept
Each pixel acts as a quantum object, its wave function collapsing under the viewer's influence—a metaphor for fluid identity in the digital age.

## Technical Implementation
- Algorithms: Wave function simulation with QMsolve.
- Interactivity: User's cursor as a collapse tool.
