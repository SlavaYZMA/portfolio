# Rajm

Rajm — это путешествие в хаотические структуры, где визуальные паттерны рождаются из случайных процессов, вдохновлённых природой.

## Концепция
Проект исследует баланс между порядком и беспорядком, используя генеративные алгоритмы.

## Техническая реализация
- Генеративное искусство: Processing и случайные функции.
- Анимация: Реальное время.

# Rajm

Rajm is a journey into chaotic structures where visual patterns emerge from random processes inspired by nature.

## Concept
The project explores the balance between order and chaos using generative algorithms.

## Technical Implementation
- Generative Art: Processing and random functions.
- Animation: Real-time.
