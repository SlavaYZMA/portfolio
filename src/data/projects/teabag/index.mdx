# Teabag

Teabag превращает обычный чайный пакетик в портал в сюрреалистический мир, где реальность смешивается с фантазией.

## Концепция
Проект исследует трансформацию банальных предметов в символы через цифровую обработку.

## Техническая реализация
- Генерация: Алгоритмы искажения.
- Интерактивность: Сенсорный ввод.

# Teabag

Teabag transforms an ordinary teabag into a portal to a surreal world where reality blends with fantasy.

## Concept
The project explores the transformation of mundane objects into symbols through digital processing.

## Technical Implementation
- Generation: Distortion algorithms.
- Interactivity: Touch input.
