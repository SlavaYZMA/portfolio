# Sensitivity

Sensitivity раскрывает внутренние эмоции через динамические цветовые градиенты, реагирующие на движение зрителя.

## Концепция
Проект преобразует субъективные чувства в визуальные формы, создавая уникальный опыт.

## Техническая реализация
- Обработка данных: Анализ движения с камеры.
- Визуализация: WebGL.

# Sensitivity

Sensitivity unveils inner emotions through dynamic color gradients that respond to the viewer's movement.

## Concept
The project transforms subjective feelings into visual forms, creating a unique experience.

## Technical Implementation
- Data Processing: Motion analysis from camera.
- Visualization: WebGL.
